Operations Dashboard

/operations/dashboard

Who can access: ManagerAdminSuperAdmin

Purpose

Live operational overview of the restaurant: sales figures, open orders, table status, kitchen state, and customer service requests in one screen.

What you see

  • Sales metrics — today's revenue and order count.
  • Open orders — currently active tickets by status.
  • Table status — which tables are occupied / free.
  • Kitchen state — pending vs in-progress items.
  • Customer requests — call-waiter / bill / water raised from the customer app.

How to use

  1. Open /operations/dashboard — data refreshes live (DB-poll bridge).
  2. Use the quick links to jump to POS, tables, reservations, or reports.
  3. Act on customer requests as they appear.

Screenshot

Tip. Table status is derived from active orders — there is no manual "seat table" step; placing/closing orders updates it.
