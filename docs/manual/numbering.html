Numbering Scopes

/admin/numbering-scopes

Who can access: AdminSuperAdmin

Purpose

Configure document number prefixes and sequences (e.g. order numbers).

How to use

  1. Open /admin/numbering-scopes.
  2. Edit a scope's prefix and sequence settings.
  3. Save — new documents use the updated format.

Screenshot

Note. Orders default to ORD-yyyyMMdd-NNNN (date-based, generated by OrderNumberGenerator).
