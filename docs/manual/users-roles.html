Users & Roles

/admin/users · /admin/roles

Who can access: AdminSuperAdmin

Purpose

Manage staff accounts and what they can do. Roles drive both feature access and which sidebar modules appear.

The six roles

  • SuperAdmin — full system, including tenants/modules.
  • Admin — full operations + administration.
  • Manager — operations, inventory, finance, reports.
  • Waiter — take and manage orders.
  • Chef — kitchen display and orders.
  • Cashier — POS, cash counter, orders.

Create / manage a user

  1. Open /admin/users → New.
  2. Enter email/username, name, and assign one or more roles.
  3. Set the initial password; the user changes it after first login.
  4. Activate/deactivate or reset password later from the same list.
